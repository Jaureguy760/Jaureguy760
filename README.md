### Jeff Jaureguy

Bioinformatics Scientist at UC San Diego and Salk Institute. Building foundation models for genomics and tools for variant interpretation.

#### Current Work

- **[evoBorzoi](https://github.com/Jaureguy760/evoBorzoi)** — Borzoi model enhanced with evolutionary priors from GPN-Star
- **[crisprFlashzoi](https://github.com/Jaureguy760/crisprFlashzoi)** — Single-cell CRISPR perturbation prediction
- **[varCA](https://github.com/Jaureguy760/varCA)** — Ensemble variant calling from ATAC-seq data
- **[t1d-graphrag-atlas](https://github.com/Jaureguy760/t1d-graphrag-atlas)** — Knowledge graph + RAG for T1D variant research
